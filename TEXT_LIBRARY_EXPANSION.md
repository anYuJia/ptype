## 📚 内容规划

### 中文现代文
**简单** (50-100字)
- 生活日常
- 简单描述
- 基础对话

**中等** (100-200字)
- 散文节选
- 新闻报道
- 故事片段

**困难** (200-300字)
- 文学作品
- 哲学思考
- 专业文章

### 中文文言文
**简单** (30-80字)
- 成语故事
- 简短寓言
- 基础古文

**中等** (80-150字)
- 古文节选
- 历史故事
- 诗词文赋

**困难** (150-250字)
- 经典篇章
- 复杂文体  
- 深度作品

### 编程语言代码库

#### Python
- 简单：基础语法、列表操作
- 中等：函数定义、类定义
- 困难：装饰器、异步编程

#### JavaScript/TypeScript
- 简单：变量、循环
- 中等：箭头函数、Promise
- 困难：泛型、高阶函数

#### Java
- 简单：类定义、方法
- 中等：接口、泛型
- 困难：设计模式、注解

#### C/C++
- 简单：变量、函数
- 中等：指针、结构体
- 困难：模板、智能指针

#### Go
- 简单：函数、切片
- 中等：goroutine、channel
- 困难：接口、反射

#### Dart/Flutter
- 简单：变量、类
- 中等：Widget、异步
- 困难：状态管理、动画

#### HTML/CSS
- 简单：基础标签
- 中等：表单、布局
- 困难：响应式、动画

#### Bash/PowerShell
- 简单：基础命令
- 中等：脚本、管道
- 困难：函数、错误处理

2. **版权问题**
   - 使用公共领域作品
   - 原创示例代码
   - 避免侵权内容

3. **用户体验**
   - 选项不要过多
   - 默认值合理
   - 切换流畅
